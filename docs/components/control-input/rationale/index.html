Control input

v3.0.2

Control inputs help users input one or more selections from multiple options. Our control inputs consist of checkboxes and radio buttons.

Size of inputs

Our checkbox and radio inputs are much larger than default browser inputs in order to provide larger hit/target areas for touchscreen devices or mobility impaired users. This was largely inspired by the work on radios and checkboxes done by gov.uk.

However we also supply a small version of the inputs for use when the large inputs are impractical.
