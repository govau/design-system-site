Get started

There are many ways to get started with the Australian Government Design System. It all depends on your project's needs and whether you are a developer or a designer (or somewhere in between). Don’t forget to check out our community to see what others are doing!

There are four ways to get started with the Design System;

  • If you have prior experience with npm, use npm to install the components
  • If you don't have prior experience with npm, visit our Download page to install the components
  • For full page templates, download the design system starter kit

Once you have the components in a website we have advanced guides for developers on how to:

  • Customise the colour scheme
  • How to use the font size and space functions
  • Customise existing components
  • Create a new component
  • Set up a development environment
  • Spacing between components

The advanced guides assume prior knowledge of npm and SASS.

Throughout the process you may notice issues, missing features or have ideas for new components. If you would like to contribute or help build the future of the Australian Government Design System you can:

  • Join the community
  • Submit a bug report
  • Submit a feature request
  • Suggest a component
